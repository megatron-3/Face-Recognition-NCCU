About Face Recognition

This is the Face Recognition Web Service. In this you can register a person and then test the face recognition system. You can make the system recognize your image from the link in the title bar.

Following are some of the things that should be kept in mind before using Face Recognition :

  1. The image which you upload should only contain 1 face otherwise the server will not recognize your face
  2. Your uploaded photo will be stored by the server
  3. Your face will be recognized using HOG technique
